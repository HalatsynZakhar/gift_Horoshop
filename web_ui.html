Подарунки Хорошоп

Масове та ручне керування подарунками товарів.

Готово до роботи

Доступ до API

Для кожної операції потрібні логін і пароль API або чинний токен. Дані не зберігаються у браузері.

Додати подарунок вручну

Вказуйте артикул, який відображається на сайті. Перед зміною сервіс звіряє обидва товари з каталогом Хорошоп.

Масові операції через Excel

Завантажити шаблон

Для масового видалення спочатку вивантажте поточні подарунки нижче, позначте потрібні рядки «Так» у стовпці «Видалити (Так)» і завантажте цей файл сюди.

1
Заповніть шаблон

Кожен рядок додає один зв'язок основного товару та подарунка.

2
Перевірте файл

Статус «Готово до операції» означає, що рядок можна передати в Хорошоп.

3
Виконайте операції

Поточні подарунки товару не стираються, змінюються тільки зазначені зв'язки.

Результат останньої операції

Після перевірки виправте лише рядки зі статусом «Помилка», а рядки «Готово до операції» можна запускати.

Ще немає результатів.
